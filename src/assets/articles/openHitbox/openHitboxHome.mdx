# openHitbox

openHit box is as it describes at hitbox style controller. It was born of my enjoyment of Tekken 8 and a the price of other
fightpads/hitboxes, often over 100 dollars. So I decided to create my own. Currently there is only one prototype () of the openHitbox.
However a version two is in the works now featuring a pcb and hotswap sockets, and I also would like to create a version that supports
[Gateron Low Profile Switches](https://www.gateron.com/collections/low-profile-series) the software side of things will be powered by
[GP2040-CE](https://gp2040-ce.info) and for the mcu a [Waveshare RP2040-Zero](https://www.waveshare.com/rp2040-zero.htm) will be used
because it has enough pins to support all of the buttons used by a hitbox. The [Waveshare RP2040-Zero](https://www.waveshare.com/rp2040-zero.htm)
also supports usb-c which is always a nice addition.
